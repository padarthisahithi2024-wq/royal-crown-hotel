# 🏨 Royal Crown Hotel Management Website

A modern and responsive hotel management website developed using HTML, CSS, and JavaScript. The website showcases luxury hotel rooms, premium services, facilities, image galleries, and an online reservation system.

---

## 📌 Project Overview

Royal Crown Hotel is a luxury hotel website designed to provide visitors with information about hotel accommodations, services, facilities, and reservations. The website offers a professional user interface with responsive design and form validation.

---

## 🚀 Features

* Responsive Navigation Bar
* Full-Screen Hero Banner
* Luxury Room Showcase
* Hotel Services Section
* Image Gallery
* Contact & Reservation Form
* JavaScript Form Validation
* Mobile-Friendly Design
* SEO Meta Tags
* Modern Luxury UI

### Reservation Form Validation

* Guest Name

* Phone Number
* Room Type

Users receive alert messages for invalid input and confirmation after successful submission.
